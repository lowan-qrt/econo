# Econo

Econo is a personal budget management application built with React.
It helps you track your expenses and income, making it easier to manage your finances daily.

## Features

- Add, edit, and delete expenses and income
- View balances and distribution charts
- Simple and responsive interface
- Designed to evolve with new features

## Requirements

Before running Econo, make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or later recommended)  
  Node.js is required to run the React development server and build the project.

- [npm](https://www.npmjs.com/get-npm) (comes with Node.js)  
  npm is the Node package manager used to install project dependencies.

You can verify the installation by running the following commands in your terminal:

```bash
node -v    # should display the Node.js version
npm -v     # should display the npm version
```

## Installation

To run Econo locally, make sure you have **Node.js** and **npm** installed.

```bash
# Clone the repository
git clone https://github.com/lowan-qrt/econo.git
cd econo

# Install dependencies
npm install

# Start the development server
npm start
```
